# `@nostdb/cli` candidate launcher

Version `0.0.3` of the dependency-free JavaScript launcher and its six
SSPL-1.0 native packages is published under `next`; `latest` remains `0.0.2`.
The launcher selects the exact-version native optional package for the current
OS and CPU, then delegates without implementing database behavior.

Tests and candidate scripts in this directory never publish by themselves. See
the repository [distribution contract](../DISTRIBUTION.md).
